# 💫 About Me:
Oi! Eu sou o Allan 👋

Sou estudante de programação e gosto de aprender construindo projetos. Tenho interesse principalmente em desenvolvimento back-end, APIs e arquitetura de aplicações, mas também gosto de explorar o front-end quando o projeto pede.

Recentemente concluí o curso Jovem Programador do Senac, onde trabalhei com tecnologias como React, Node.js e autenticação com JWT. Desde então continuo estudando por conta própria, criando projetos e resolvendo problemas para evoluir minhas habilidades. Também estudo Python de forma independente e gosto de testar novas ferramentas e ideias.

Atualmente tenho explorado coisas como APIs com NestJS, ORMs como Prisma, validação de dados e diferentes formas de estruturar aplicações. Também gosto de experimentar com novas tecnologias e bibliotecas — seja para melhorar um projeto ou apenas por curiosidade.

Aqui no GitHub você vai encontrar projetos que refletem meu processo de aprendizado. Estou sempre aberto a trocar ideias, aprender coisas novas e colaborar em projetos interessantes.
